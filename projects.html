Expense Tracker

×
  • The expense tracker is designed to track expenses in a systematic way instead of manual tracking.
  • It allows you to check daily, monthly, and yearly data with bar chart references.
  • A built-in chatbot helps users retrieve specific details for a given date, month, or year, along with features like viewing remaining balance, changing names, editing salary, etc.
  • A to-do list is included to note daily tasks and monitor their status as done or not done.
  • For security, Firebase is used to store user details securely, including email OTP verification.

Web App Builder

×
  • The Application Builder is specially created for those who are learning to code.
  • It provides basic app templates along with their source code so that users can make changes and learn by doing.
  • There are 10 pre-built templates: Calculator App, Tic-Tac-Toe, Resume Builder, Portfolio Website, Note Taking App, Business Website, To-Do List, Stopwatch & Timer, Drawing App, Quiz App (Tamil)
  • Users can download and use the apps locally.
  • For secure login, Firebase with email authentication is used.
